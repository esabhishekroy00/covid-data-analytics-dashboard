# Global COVID-19 Data Analytics Project

## Tools
Python, Pandas, SQL, Matplotlib

## Project Overview
This project analyzes global COVID-19 data to identify trends in infection rates, death rates, and country-level impact.

## Features
- Data cleaning using Python
- Exploratory Data Analysis
- SQL analytics queries
- Data visualization

## Dataset
COVID-19 dataset from Kaggle.

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. SQL Query Analysis
5. Visualization
